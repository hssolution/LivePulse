# 🎯 서비스 기획서

## 서비스 개요

### 서비스명
(미정) - 오프라인 강좌 실시간 상호작용 플랫폼

### 서비스 컨셉
**오프라인 강좌에서 강사와 청중이 실시간으로 상호작용하는 플랫폼**

기존 오프라인 강좌의 일방향 소통 한계를 극복하고, 청중의 적극적인 참여를 유도하여 강좌의 효과를 극대화합니다.

---

## 핵심 기능

### 1. 실시간 설문/투표 (Live Poll)
- 강사가 설문/투표를 생성하면 청중이 모바일로 즉시 참여
- 결과가 실시간으로 강사 화면에 차트/그래프로 표시
- 다양한 설문 유형 지원:
  - 객관식 (단일/다중 선택)
  - 주관식
  - 평점/별점
  - 워드 클라우드
  - O/X 퀴즈

### 2. 실시간 Q&A (Live Questions)
- 청중이 모바일로 질문 제출
- 질문이 실시간으로 강사 화면에 표시
- 다른 청중이 질문에 '좋아요' 투표 가능 (인기 질문 상단 노출)
- 강사가 질문을 선택하여 답변 표시
- 익명/실명 선택 가능

### 3. 실시간 피드백 (Live Feedback)
- 강좌 진행 중 청중의 이해도/만족도 실시간 체크
- 이모지 반응 (👍 👎 ❓ 💡 👏)
- "속도 조절" 피드백 (너무 빠름 / 적당함 / 너무 느림)

### 4. 세션 관리
- 강사가 세션(강좌) 생성
- QR코드 또는 세션 코드로 청중 참여
- 세션별 참여자 수, 참여율 통계
- 세션 종료 후 결과 리포트 생성

### 5. 자료 공유
- 강좌 자료(PDF, 이미지) 실시간 공유
- 청중이 모바일에서 자료 확인
- 북마크/메모 기능

## 사용자 유형

### 1. 강사 (Presenter/Host)
- 세션 생성 및 관리
- 설문/투표 생성 및 진행
- 실시간 질문 확인 및 답변
- 통계 및 리포트 확인
- **사용 환경**: 데스크톱/태블릿 (큰 화면)

### 2. 청중 (Audience/Participant)
- 세션 참여 (QR코드/세션코드)
- 설문/투표 참여
- 질문 제출 및 투표
- 피드백 전송
- **사용 환경**: 모바일 (스마트폰)

### 3. 관리자 (Admin)
- 전체 사용자 관리
- 강사 계정 승인
- 시스템 설정
- 통계 및 분석

## 화면 구성

### 강사용 화면 (Desktop/Tablet)

#### 1. 대시보드
- 진행 중인 세션 현황
- 최근 세션 목록
- 통계 요약 (총 세션, 총 참여자, 평균 참여율)

#### 2. 세션 진행 화면 (Presentation Mode)
- **메인 영역**: 현재 진행 중인 콘텐츠 (설문 결과, 질문 목록 등)
- **사이드바**: 세션 컨트롤 (다음 설문, 질문 선택 등)
- **하단바**: 참여자 수, 실시간 피드백 요약
- **풀스크린 모드**: 프로젝터 출력용

#### 3. 세션 관리
- 세션 생성/수정/삭제
- 설문/질문 미리 준비
- 세션 설정 (익명 허용, 참여 제한 등)

#### 4. 리포트
- 세션별 상세 리포트
- 설문 결과 분석
- 질문 목록 및 답변
- PDF/Excel 내보내기

### 청중용 화면 (Mobile)

#### 1. 세션 참여
- QR코드 스캔 또는 세션코드 입력
- 닉네임 입력 (익명 가능)

#### 2. 메인 화면
- 현재 진행 중인 콘텐츠에 따라 동적 변경
- 설문 참여 / 질문 제출 / 피드백 전송
- 간단하고 직관적인 UI

#### 3. 질문 목록
- 다른 청중의 질문 확인
- 좋아요 투표
- 내 질문 확인

## 기술 스택 (예정)

### Frontend
- React.js + Vite
- Tailwind CSS + shadcn/ui
- Socket.io (실시간 통신)

## 디자인 방향

### 컨셉
- **실시간성 강조**: 라이브 인디케이터, 실시간 업데이트 애니메이션
- **직관적인 UI**: 강좌 중 빠르게 조작 가능
- **모바일 최적화**: 청중용 화면은 모바일 퍼스트
- **다크 모드 지원**: 강연장 조명에 맞게 선택 가능

### 색상
- **Primary**: 에너지 있고 활발한 느낌 (오렌지, 코랄, 또는 그린 계열)
- **Accent**: 실시간 알림, 라이브 표시용 (레드, 핑크)
- **Background**: 클린하고 집중할 수 있는 배경

### 참고 서비스
- [Slido](https://www.slido.com/) - 실시간 Q&A, 투표
- [Mentimeter](https://www.mentimeter.com/) - 인터랙티브 프레젠테이션
- [Kahoot](https://kahoot.com/) - 게임형 퀴즈
- [Pigeonhole Live](https://pigeonholelive.com/) - Q&A, 설문

### UI/UX 원칙
1. **3초 규칙**: 모든 주요 액션은 3초 이내에 완료 가능
2. **큰 터치 타겟**: 모바일에서 쉽게 탭 가능한 버튼 크기
3. **실시간 피드백**: 모든 액션에 즉각적인 시각적 피드백
4. **에러 최소화**: 강좌 중 에러 발생 시 빠른 복구
